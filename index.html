  1. git clone (cloned url) = clones the git repository at the current directory
  2. git status = checks to see if the online git is the same as current directory. If its different, a file will appear red.
  3. git add . = adds the current directory for moving to git. will make files appear green in git status.
  4. git commit -m "message for the commit" = commits changes with message
  5. git push = move the directory onto the online git repository.
